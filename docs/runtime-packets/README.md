# Runtime packets (dev-method)

Templates and executed records for **CONTROL PLANE** runtime gates. Docs-only in this repo.

- **Templates:** `templates/` — copy and fill after real execution; never pre-mark PASS/FAIL during prep batches.
- **Executed:** (optional) `executed/` — dated copies filed after home-machine runs. Not created by preparation-only work.

See `docs/control-plane/operating-memory.md` for gate chain context.
